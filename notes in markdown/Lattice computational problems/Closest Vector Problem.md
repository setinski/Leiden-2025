>[!define] #definition
>##### CVPs
>The approximate Closest Vector Problem with approximation factor $\alpha \geq 1$, denoted ==$\alpha$-CVP==, is defined as follows:
>- Given as input the basis $\mathbf{B}$ of a lattice $\mathcal{L} \subseteq \mathbb{R}^{n}$ and a target $\mathbf{t} \in \mathbb{R}^{n}$
>- Output $\mathbf{v} \in L$ such that $\| \mathbf{v} -\mathbf{t} \| \leq \alpha d(\mathbf{t},L)$.
>
>==ExactCVP== is by definition $1$-SVP.
